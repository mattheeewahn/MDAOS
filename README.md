Welcome and thank you to everyone who visited my repository.

This is MDAOS, which is an OS for an MDA 80486 homebrew computer.
The minimum requirement is an i386 processor or above.
You can run this OS on QEMU using WSL.

For example, you can:

cd "/mnt/c/Users/withd/Desktop/#######/##### n"
ls
qemu-system-i386 -fda floppy.img -boot a

Then the OS will boot, and you can experience a true retro OS on your computer.
Also, this OS is very light and can be booted from a floppy disk (1.44MB), so you can use it as a test OS for your homebrew computer.

Since the main kernel is 14KB, you can change, fix, and customize it as you want.
I welcome you to modify and distribute my OS. However, please make sure to include my GitHub username or "Matthew Ahn" in the README file.

Thank you and enjoy!

Cheers.
